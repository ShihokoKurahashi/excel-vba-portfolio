概要

本ツールは、Excelの設定シート（Config）に基づき、
テンプレート行を指定件数複製し、CSVファイルを自動生成する業務支援ツールです。

既存のブラックボックス化したVBAツールをリファクタリングし、
可読性・保守性・再利用性を意識した構造へ再設計しました。

 開発背景
 • 既存コードが長大かつ責務が曖昧で、修正時の影響範囲が不明確
 • 設定値が散在しており、仕様変更に弱い構造
 • フォルダ管理が手動で属人化していた

これらの課題を解消するため、構造の再整理とモジュール分割を実施しました。

設計上の工夫

1. 構造体による設定管理（カプセル化）

Private Type AppConfig を定義し、設定値を一元管理。

目的
 • マジックナンバー排除
 • 参照箇所の明確化
 • 仕様変更への柔軟対応

2. 単一責任を意識した処理分割

Main をフロー管理専用とし、以下へ責務分離：
 • LoadConfig
 • GenerateTemplateRows
 • ExportCSV
 • CreateFolderIfNotExists

効果
 • 可読性向上
 • バグ切り分け容易化
 • 将来的な機能追加に対応しやすい構造

3. パフォーマンス考慮
 • 最終行判定後の一括削除で高速化
 • 出力列を設定化し、レイアウト変更に対応可能


4. 属人化を防ぐファイル管理設計
 • 実行日時＋クライアント名でフォルダ自動生成
 • 実行ログ（開始・終了時刻）を記録

 使用技術
 • VBA（Excel）
 • CSVファイル出力
 • フォルダ自動生成
 • テンプレート制御ロジック

 今後の改善案
 • 配列経由での高速CSV出力
 • FileSystemObjectによる堅牢なフォルダ生成
 • 包括的なエラーハンドリング強化
