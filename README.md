概要

本ツールは、Excelの設定シート（Config）に基づき、<br>
テンプレート行を指定件数複製し、CSVファイルを自動生成する業務支援ツールです。<br>

既存のブラックボックス化したVBAツールをリファクタリングし、
可読性・保守性・再利用性を意識した構造へ再設計しました。<br>

 開発背景<br>
 • 既存コードが長大かつ責務が曖昧で、修正時の影響範囲が不明確<br>
 • 設定値が散在しており、仕様変更に弱い構造<br>
 • フォルダ管理が手動で属人化していた<br>

これらの課題を解消するため、構造の再整理とモジュール分割を実施しました。<br>

設計上の工夫<br>
1. 構造体による設定管理（カプセル化）<br>
Private Type AppConfig を定義し、設定値を一元管理。<br>

目的<br>
 • マジックナンバー排除<br>
 • 参照箇所の明確化<br>
 • 仕様変更への柔軟対応<br>

2. 単一責任を意識した処理分割<br>
Main をフロー管理専用とし、以下へ責務分離：<br>
 • LoadConfig<br>
 • GenerateTemplateRows<br>
 • ExportCSV<br>
 • CreateFolderIfNotExists<br>

効果<br>
 • 可読性向上<br>
 • バグ切り分け容易化<br>
 • 将来的な機能追加に対応しやすい構造<br>

3. パフォーマンス考慮<br>
 • 最終行判定後の一括削除で高速化<br>
 • 出力列を設定化し、レイアウト変更に対応可能<br>

4. 属人化を防ぐファイル管理設計<br>
 • 実行日時＋クライアント名でフォルダ自動生成<br>
 • 実行ログ（開始・終了時刻）を記録<br>

 使用技術<br>
 • VBA（Excel）<br>
 • CSVファイル出力<br>
 • フォルダ自動生成<br>
 • テンプレート制御ロジック<br>

 今後の改善案<br>
 • 配列経由での高速CSV出力<br>
 • FileSystemObjectによる堅牢なフォルダ生成<br>
 • 包括的なエラーハンドリング強化<br>
